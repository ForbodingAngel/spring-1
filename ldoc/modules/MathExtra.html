math.hypot(x, y)
Returns the length of hypotenuse of right angle triangle with sides x and y, equivalent to sqrt(xx + yy), but has better numerical stability and internally handles intermediate overflows/underflows, but is also slower.

Parameters:

  • x number
  • y number

Returns:

    number sqrt(xx+yy)
math.diag(x1[, x2[, x3[, xn]]])
Returns the length of the diagonal of an n-dimensional box (or the length of an n-component vector). Rather quick method that does not handle intermediate overflows/underflows nor is made for numerical stability.

Parameters:

  • x1 number
  • x2 number (optional)
  • x3 number (optional)
  • xn number and so on (optional)

Returns:

    number diagonal

math.round(x, decimals)
Returns x rounded to n decimals, if n is omitted or <=0, rounds to nearest integer. Note that Spring's Lua interpreter uses 32-bit floats for all numbers so max. precision is ~7 decimal digits.

Parameters:

  • x number
  • decimals number

Returns:

    number rounded
